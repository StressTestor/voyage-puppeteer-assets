# voyage-puppeteer-assets

public asset host for Voyage worlds built via [World-Puppeteer](https://github.com/nikolaj-lat/World-Puppeteer).

URLs are referenced from `config.json` files inside Voyage worlds. raw URL pattern:

```
https://raw.githubusercontent.com/StressTestor/voyage-puppeteer-assets/main/portraits/<slug>.png
```

## structure

```
portraits/   character portraits, named by slug
```

generated via Flux 2 Pro through OpenRouter, ~1024x1024 painterly digital art.
